# HRMS Lite
Lightweight HR management system for employee and attendance tracking.

## Tech Stack
- [https://hrms-lite-omega-three.vercel.app/] Frontend: React (Vite) [cite: 91]
- [https://hrms-backend-ulon.onrender.com/] Backend: FastAPI (Python) [cite: 93]

## Setup
1. Backend: `pip install -r requirements.txt` then `uvicorn main:app --reload`
2. Frontend: `npm install` then `npm run dev`
